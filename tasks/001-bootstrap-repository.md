# Task 001 — Bootstrap the Repository

## Objective

Create the minimal repository structure and quality gates without implementing business workflows.

## Scope

- backend module skeleton
- frontend module skeleton
- shared contract module
- architecture-test module
- local development environment
- unit-test framework
- linting and formatting
- dependency and secret scanning
- pull-request checks
- health endpoint
- structured logging with redaction support

## Constraints

- no production credentials
- no real Oracle integration
- no cloud-specific deployment
- no business logic
- no AI model calls
- synthetic data only

## Deliverables

- build succeeds locally
- test suite runs
- CI configuration validates formatting, tests, and scans
- README explains local setup
- architecture tests prevent forbidden module dependencies
- dependency inventory is generated

## Pull-request requirement

Open a draft pull request. Do not merge.
