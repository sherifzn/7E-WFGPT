# Architecture Decisions Required Before Production Coding

Create one ADR for each item:

1. Core workflow runtime engine
2. BRE and DMN implementation
3. Primary backend language: Java or Kotlin
4. Frontend framework and design system
5. Event-bus technology
6. Runtime database
7. Document and evidence storage
8. Identity federation implementation
9. Oracle integration pattern
10. AI Gateway and model-routing approach
11. Knowledge retrieval and vector indexing
12. Cloud, hybrid, or on-premises deployment
13. Container orchestration platform
14. Secrets and key management
15. Observability stack
16. Disaster-recovery objectives
17. Data-residency requirements
18. Source-code and coding-agent privacy controls

Until approved, keep implementation behind explicit interfaces.
